# “45分钟到8分钟”提示语正式化设计

## 目标

将“高级感不靠堆图，而是让‘45→8’这条链路一眼看懂”替换为正式、客观的数据新闻表述，移除面向设计过程的元话语。

## 方案比较

1. **数据结果型（采用）**：直接说明设备改进、时间变化和效率结果，信息最完整、语气最正式。
2. **过程说明型**：强调地面操作与快速复核，但会重复相邻流程卡片。
3. **简短结论型**：只写“作业时间缩短约82.2%”，但会重复上方黄色数据标签。

## 最终文案

> 可摘取式低压刀闸将保险更换时间由45分钟缩短至8分钟，显著提升了抢修效率。

## 范围与验证

- 仅修改 `index.html` 中“现场改造”提示卡的正文。
- 不调整样式、结构或其他文案。
- 验证旧文案在线上消失，新文案完整显示。
